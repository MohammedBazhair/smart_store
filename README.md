# Smart Store - تطبيق إدارة المنتجات

تطبيق Flutter احترافي لإدارة المنتجات مع مسح الباركود ونظام الصلاحية والتنبيهات.

## المميزات

### 🎯 الميزات الرئيسية

- **مسح الباركود**: دعم EAN-13 و UPC مع حفظ الأسعار تلقائيًا
- **إدارة المنتجات**: إضافة، تعديل، حذف، وبحث عن المنتجات
- **نظام الصلاحية**: تتبع تواريخ انتهاء المنتجات مع تصنيف تلقائي
- **التنبيهات**: تنبيهات ذكية قبل 30 يوم، 7 أيام، ويوم واحد من الانتهاء
- **لوحة التحكم**: عرض إحصاءات شاملة للمنتجات
- **نظام العملات**: دعم الريال اليمني (YER) والريال السعودي (SAR) مع تحويل تلقائي
- **النسخ الاحتياطي**: إنشاء واستعادة النسخ الاحتياطية

### 🏗️ المعمارية

- **Clean Architecture**: فصل واضح بين الطبقات (Presentation, Domain, Data)
- **SOLID Principles**: كود نظيف وقابل للصيانة
- **Modular Design**: ميزات منفصلة وقابلة لإعادة الاستخدام

### 🛠️ التقنيات المستخدمة

- **State Management**: Riverpod (Generator)
- **Database**: SQLite (sqflite)
- **Barcode Scanner**: mobile_scanner
- **Notifications**: flutter_local_notifications
- **UI**: Material Design 3 مع تصميم احترافي

## التثبيت

1. استنساخ المشروع:
```bash
git clone <repository-url>
cd smart_store
```

2. تثبيت الحزم:
```bash
flutter pub get
```

3. تشغيل code generation:
```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

4. تشغيل التطبيق:
```bash
flutter run
```

## الاستخدام

### إضافة منتج جديد

1. اضغط على زر "إضافة منتج" في شاشة المنتجات
2. أدخل معلومات المنتج (الاسم، الكمية، تاريخ الانتهاء، الفئة)
3. يمكنك مسح الباركود أو إدخاله يدويًا
4. احفظ المنتج

### مسح الباركود

1. اضغط على زر "مسح الباركود" من الشاشة الرئيسية
2. وجه الكاميرا نحو الباركود
3. إذا كان السعر محفوظًا، سيظهر تلقائيًا
4. إذا لم يكن محفوظًا، سيُطلب منك إدخال السعر

### إعدادات العملة

1. انتقل إلى الإعدادات
2. اختر العملة الافتراضية (YER أو SAR)
3. حدد سعر الصرف (1 SAR = ? YER)
4. سيتم التحويل تلقائيًا عند العرض

### النسخ الاحتياطي

1. انتقل إلى الإعدادات
2. اضغط على "إنشاء نسخة احتياطية" لحفظ البيانات
3. اضغط على "استعادة نسخة احتياطية" لاستعادة البيانات
