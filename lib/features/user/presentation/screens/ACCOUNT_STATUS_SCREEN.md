# شاشة حالة الحساب (Account Status Screen)

## نظرة عامة
شاشة احترافية وجذابة لعرض حالة حساب المستخدم عند أول فتح للتطبيق بعد تسجيل الدخول. تتميز بتصميم UI/UX حديث مع رسوم متحركة سلسة وألوان ديناميكية تتغير حسب حالة الحساب.

## الميزات الرئيسية

### 1. التصميم المرئي
- **خلفية متدرجة** تتغير حسب حالة الحساب
- **أيقونة دائرية متحركة** مع حلقات توسعية نابضة
- **بطاقات معلومات** بظلال ناعمة وحواف مستديرة
- **رسوم متحركة** سلسة عند ظهور العناصر (Fade, Scale, Slide)

### 2. حالات الحساب المدعومة

#### ✅ حساب نشط (Active)
- **اللون**: أخضر (#059669)
- **الأيقونة**: `verified_user_rounded`
- **الميزات**:
  - الوصول الكامل لجميع الميزات
  - إمكانية إضافة وإدارة المنتجات
  - تلقي الإشعارات والتنبيهات
  - الدعم الفني على مدار الساعة

#### ⏸️ حساب مجمد (Frozen)
- **اللون**: برتقالي (#D97706)
- **الأيقونة**: `pause_circle_outline_rounded`
- **الميزات**:
  - إمكانية تسجيل الدخول والتصفح
  - عرض البيانات والمعلومات
  - بعض الميزات محدودة مؤقتاً
  - التواصل مع الدعم الفني متاح

#### ⏳ في انتظار التفعيل (Pending)
- **اللون**: بنفسجي (#6366F1)
- **الأيقونة**: `hourglass_empty_rounded`
- **الميزات**:
  - الحساب قيد المراجعة
  - سيتم التفعيل قريباً
  - يمكنك تصفح التطبيق
  - ستصلك إشعارات عند التفعيل

### 3. المعلومات المعروضة
- اسم المستخدم
- رقم الهاتف (إن وُجد)
- دور المستخدم (مدير/مستخدم/زائر)
- تاريخ إنشاء الحساب
- حالة الحساب التفصيلية
- قائمة بالميزات المتاحة

## كيفية الاستخدام

### الاستخدام الأساسي
```dart
AccountStatusScreen(
  profile: userProfile,
  onContinue: () {
    // الانتقال إلى الشاشة الرئيسية
    Navigator.pushReplacement(
      context,
      MaterialPageRoute(builder: (_) => DashboardScreen()),
    );
  },
)
```

### التكامل مع AuthGate
الشاشة مدمجة تلقائياً مع `AuthGate` وتظهر:
- عند أول فتح للتطبيق بعد تسجيل الدخول
- مرة واحدة فقط (يتم حفظ الحالة في SharedPreferences)
- يمكن إعادة عرضها بحذف المفتاح `has_shown_account_status`

## التخصيص

### تغيير الألوان
يمكنك تخصيص الألوان لكل حالة في دالة `_getStatusConfig`:

```dart
case AccountStatus.active:
  return _StatusConfig(
    primaryColor: Colors.green,  // اللون الأساسي
    secondaryColor: Colors.lightGreen,  // اللون الثانوي
    // ...
  );
```

### تخصيص الرسوم المتحركة
يمكنك تعديل مدة وسرعة الرسوم المتحركة:

```dart
_animationController = AnimationController(
  vsync: this,
  duration: const Duration(milliseconds: 1200),  // المدة الإجمالية
);
```

### إضافة ميزات جديدة
لإضافة ميزات لحالة معينة:

```dart
features: [
  'ميزة جديدة 1',
  'ميزة جديدة 2',
  // ...
],
```

## الرسوم المتحركة

### 1. Fade Animation
- **المدة**: 0-60% من الوقت الكلي
- **التأثير**: ظهور تدريجي للشاشة

### 2. Scale Animation
- **المدة**: 20-80% من الوقت الكلي
- **المنحنى**: `Curves.elasticOut`
- **التأثير**: تكبير الأيقونة بشكل مرن

### 3. Slide Animation
- **المدة**: 30-100% من الوقت الكلي
- **المنحنى**: `Curves.easeOutCubic`
- **التأثير**: انزلاق العناصر من الأسفل

### 4. Pulsing Rings
- **التكرار**: 3 حلقات
- **المدة**: 2-3.5 ثانية
- **التأثير**: حلقات متوسعة تتلاشى

## ملاحظات مهمة

1. **الأداء**: الشاشة محسّنة للأداء مع استخدام `SingleTickerProviderStateMixin`
2. **الذاكرة**: يتم التخلص من `AnimationController` تلقائياً في `dispose()`
3. **التوافق**: تدعم اللغة العربية بالكامل مع RTL
4. **الاستجابة**: تتكيف مع جميع أحجام الشاشات

## إعادة تعيين الحالة

لإعادة عرض الشاشة مرة أخرى:

```dart
final prefs = await SharedPreferences.getInstance();
await prefs.remove('has_shown_account_status');
```

أو من خلال مسح جميع البيانات:

```dart
await prefs.clear();
```
